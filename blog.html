Question-1: What are the different ways to select an element in the DOM?

You can select elements in the DOM using methods like getElementById("id") for a single element and getElementsByClassName("class") or getElementsByTagName("tag") for multiple elements. More flexible options include querySelector("css-selector") for the first match and querySelectorAll("css-selector") for all matches.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML gets or sets the HTML content inside an element, including tags, while innerText gets only the visible text, ignoring hidden elements . textContent gets all text inside an element, including hidden elements, but treats everything as plain text without formatting.

Question-3: What is event delegation in the DOM?

Event delegation means adding one event listener to a parent instead of many to each child. When a child is clicked, the event "bubbles up" to the parent, which handles it. This makes the webpage faster and works even if new child elements are added later.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts from the clicked (or interacted) element and moves up to its parent, grandparent, and so on. For example, if you click a button inside a <div> , the event first happens on the button, then "bubbles up" to the <div>, and then to the <body>. This allows parent elements to handle events from their children using event delegation.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, use document.createElement("tag"), like document.createElement("div"). To add it, use parent.appendChild(element), which places it inside another element. To remove it, use element.remove() or parent.removeChild(element).
